spancopy Package Repository

APT Repository

Add to /etc/apt/sources.list.d/spancopy.list:

deb [trusted=yes] https://iamazeem.github.io/spancopy/packages/apt/\$(dpkg --print-architecture)/ ./

RPM Repository

Add to /etc/yum.repos.d/spancopy.repo:

[spancopy] name=SpanCopy Repository baseurl=https://iamazeem.github.io/spancopy/packages/rpm/\$basearch enabled=1 gpgcheck=0
